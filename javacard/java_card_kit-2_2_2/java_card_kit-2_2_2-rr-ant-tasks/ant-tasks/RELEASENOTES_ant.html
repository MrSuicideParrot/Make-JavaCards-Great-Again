Introduction

The development kit tools for the Java CardTM platform version 2.2.2 require the use of Apache Ant. However, Ant's use in the tools is not apparent to users. See Development Kit User's Guide for information on the tools and demos and on installing Ant. Once you install Ant, Ant's use in the development kit will not be apparent because Ant is simply used by the development kit to run the tools and demos.

However, the Ant tasks in the Ant tasks bundle are not part of the development kit and their use is optional and unsupported.  They are provided simply for Ant users who might find they make using the development kit tools more efficient.  Therefore, the use of Ant as described in the Ant Tasks User's Guide is unsupported.

Installation Instructions

If you do not already have Apache Ant installed on your system, download version 1.6.2 of Apache Ant from their web site at http://ant.apache.org. Unzip the package into a directory other than the development kit's directory. Add Apache Ant's bin directory to your system path.  See the Development Kit User's Guide or Ant Tasks User's Guide for more information on Apache Ant installation and setup.

The Ant tasks bundle unzips within the binary release bundle at java_card_kit-2_2_2/ant-tasks.  When you unzip the Ant tasks bundle, the Ant tasks' JavaTM Archive (JAR) file is extracted into the subdirectory java_card_kit-2_2_2/ant-tasks/lib/jctasks.jar. The user's guide for the Ant tasks is extracted into the subdirectory java_card_kit-2_2_2/ant-tasks/docs. For more information, see Ant Tasks User's Guide.

NOTE: The use of Ant is supported only for use with the development kit. The use of Ant outside the development kit is not supported. The Ant tasks have not been thoroughly tested.

Documentation

Use of the Ant tasks within the development kit will not be apparent as you use the tools and demos. However, for documentation on the unsupported use of the Ant tasks outside the development kit, see the Ant Tasks User's Guide in the subdirectory ant-tasks/docs.

The Ant Tasks User's Guide is provided in HTML and PDF formats. In addition, the JavadocTM tool files for the Ant tasks are in the subdirectory java_card_kit-2_2_2/ant-tasks/docs/html/javadocs. A compilation of the Javadoc tool files in PDF format is in the subdirectory java_card_kit-2_2_2/ant-tasks/docs/pdf/ant-tasks-javadocs.pdf.

HTML can be viewed with any CSS-compliant browser, such as:

  • Netscape Communicator, version 5.0 or later
  • Mozilla, version 1.1 or later

PDF files can be viewed in your web browser with an appropriate plugin or in Adobe® Acrobat Reader. Most recent browsers include the PDF reader plugin. If your browser does not, you can download the plugin from the browser vendor's web site or the Adobe web site at:

http://www.adobe.com/products/acrobat/readstep.html

Documentation files packaged in the release were current when the release shipped. See the Java Card technology web site at http://java.sun.com/products/javacard/ for any updates.
